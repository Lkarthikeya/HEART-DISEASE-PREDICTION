 #  Heart Disease Prediction Web App

##  Overview
This project predicts the risk of heart disease using machine learning.
Users can enter medical details through a web interface and get real-time predictions.


##  Features
- Predict heart disease risk (High / Low)
- Shows probability score
- Clean web interface using Streamlit
- Machine learning model using Logistic Regression


## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit


##  Dataset
- File: `heart.csv`
- Features: 13 medical attributes
- Target:
  - 1 → Heart Disease
  - 0 → No Disease
